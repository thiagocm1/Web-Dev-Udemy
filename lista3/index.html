O que é document?

O Document object representa sua página, na qual o DOM Object, faz com que seja possivel o acesso de qualquer elemento presente em uma página HTML

Explique as funções da API do DOM listadas abaixo, indicando sua assinatura e funcionalidade.

  • getElementById

    Procura um elemento por uma id, como essa aqui com a cor azul no css! ou no javascritp!

  • getElementByClassName

    Procura o DOM Object representado por uma classe!

  • querySelector

    querySelector() é um metodo na qual retorna o primeiro elemento que é especificado para o seletor.

  • QuerySelectorAll

    Retorna uma lista de elementos presentes no documento (usando ordenação em profundidade, pré-ordenada e transversal dos nós do documento) que coincidam com o grupo de seletores especificado

  • appendChild

  • getAttribute()

    getAttribute() retorna o valor de um argumento específico do elemento. Se o atributo não existir, o valor retornado será null

    A propriedade Element.innerHTML define ou obtém a sintaxe HTML descrevendo os elementos descendentes.
